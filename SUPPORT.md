# UCX Browser Support

## Getting Help
https://github.com/KGECMD/ucx-browser/issues

## Common Issues

### SmartScreen Warning
Click "More info" -> "Run anyway". UCX is open source and safe.

### Extensions Not Installing
Enable Chrome Web Store access in settings.

### VPN Issues
Configure PlanetVPN extension with your credentials.

### Build Issues
See docs/building.md for build instructions.

## Platform Support
- Windows 7, 10, 11 (x64)
- Linux (x64, ARM64)
- macOS (Intel and Apple Silicon)
